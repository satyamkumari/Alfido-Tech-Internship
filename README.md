# 📊 Data Analytics & Business Intelligence Projects

This repository contains multiple data analytics and business intelligence projects focused on customer behavior, sales performance, and website traffic analysis. These projects demonstrate practical applications of data analysis, visualization, and business intelligence techniques using Python, Jupyter Notebook, and Power BI.

---

# 📌 Projects Overview

## 🧾 Task 1 — Customer Behavior Analysis

### 📂 Dataset
https://www.kaggle.com/datasets/bhanupratapbiswas/customer-behavior-analysis

### 🎯 Goal
Analyze customer transactions and behavior to identify customer segments, purchase patterns, and churn risks.

### ✅ Requirements
- Perform data cleaning and feature engineering
- Segment customers using RFM analysis or clustering
- Visualize customer purchase behavior and retention trends
- Provide actionable business recommendations

### 📈 Key Analysis Performed
- Customer segmentation
- RFM analysis
- Purchase frequency analysis
- Churn risk identification
- Customer retention trends
- Spending behavior analysis

### 📦 Deliverables
- Jupyter Notebook with code and visualizations
- PDF report containing findings and recommendations

---

## 📊 Task 2 — Sales Performance Analysis

### 📂 Dataset
https://www.kaggle.com/datasets/bhanupratapbiswas/superstore-sales

### 🎯 Goal
Analyze sales performance across regions, categories, and time periods. Identify best and worst-performing products along with seasonal sales trends.

### ✅ Requirements
- Clean and prepare sales data
- Create KPIs such as revenue and average order value
- Build visualizations for trends and regional analysis
- Identify top-performing products
- Suggest tactical business improvements

### 📈 Key Analysis Performed
- Sales trend analysis
- Regional sales comparison
- Product performance analysis
- Category-wise sales analysis
- Customer segment analysis
- Seasonality analysis
- Interactive Power BI dashboard creation

### 📦 Deliverables
- Jupyter Notebook with analysis and charts
- Power BI/Tableau dashboard or PDF summary

---

## 🌐 Task 3 — Website Traffic Analysis

### 📂 Dataset
https://www.kaggle.com/datasets/bhanupratapbiswas/website-traffic-analysis

### 🎯 Goal
Analyze website traffic logs to understand user journeys, landing pages, bounce rates, and referral sources.

### ✅ Requirements
- Parse and clean traffic/log data
- Compute traffic metrics and engagement statistics
- Visualize user behavior and navigation flows
- Recommend optimizations to improve conversions

### 📈 Key Analysis Performed
- Session analysis
- User behavior tracking
- Bounce rate analysis
- Average session duration analysis
- Referral source analysis
- Landing and exit page analysis
- Conversion optimization insights

### 📦 Deliverables
- Jupyter Notebook with visualizations
- Short report with findings and recommendations
